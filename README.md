# Task Manager using NestJS

## Description

Task Manager Backend using NestJS

## Technologies/Dependencies used

- NestJS
- PostgreSQL
- TypeORM
- PassportJS
- JWT

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
